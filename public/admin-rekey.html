Secure Data Passphrase

Admin Only
Panduan Singkat
Langkah 1: Enkripsi semua file plaintext terlebih dahulu (sekali saja) dengan tombol "Encrypt Plaintext".
Langkah 2: Ganti passphrase (Rekey). Setelah berhasil, restart aplikasi dengan passphrase baru.
Catatan: Jika Anda lupa passphrase baru, data tidak bisa dipulihkan. Simpan dengan aman.
Encrypt Plaintext Files

Menulis ulang semua file data yang belum terenkripsi menjadi terenkripsi menggunakan passphrase saat ini.

Restore Backup

Pilih file backup (.enc terenkripsi atau .json). Sistem akan mendeteksi otomatis dan memulihkan data.
